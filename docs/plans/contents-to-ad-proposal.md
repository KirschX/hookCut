아래는 실제 과제 제출용 기획서에 가깝게 다시 정리한 버전이다.
핵심은 **“AI 이미지 생성기”가 아니라 “콘텐츠를 광고 소재로 바꾸는 광고 기획 워크플로우 제품”** 으로 정의하는 것이다.

---

# HookCut 기획서

## 콘텐츠를 넣으면, AI가 광고 소재로 바꿔주는 Contents-to-Ads 서비스

---

## 1. 제품 한 줄 정의

**HookCut은 소설 본문, 위키 URL, 설정 문서 등 다양한 콘텐츠 자료를 입력하면, AI가 스토리의 광고 포인트를 분석하고 사용자에게 필요한 선택지만 질문한 뒤, 광고 배너/컷 이미지 소재를 생성해주는 서비스다.**

---

# 2. Why

## 2-1. 풀고자 하는 문제

웹소설, 캐릭터 IP, 스토리형 콘텐츠는 홍보가 어렵다.
콘텐츠 자체는 길고 복잡하지만, 광고 소재는 짧고 직관적이어야 한다.

창작자나 소규모 운영자가 광고 소재를 만들 때 겪는 문제는 다음과 같다.

1. **무엇을 강조해야 할지 모른다**

   * 주인공?
   * 갈등?
   * 반전?
   * 로맨스?
   * 세계관?
   * 초반 후킹 장면?

2. **어떤 타겟에게 어떻게 보여줘야 할지 어렵다**

   * 10대 여성향 독자에게 먹히는 소재와
   * 30대 남성 판타지 독자에게 먹히는 소재는 다르다.

3. **범용 이미지 생성기는 광고 기획을 대신해주지 않는다**

   * 프롬프트를 직접 써야 한다.
   * 어떤 장면을 이미지로 만들지 사용자가 정해야 한다.
   * 광고 목적, 타겟, 카피, 컷 구성은 별도로 고민해야 한다.

4. **여러 버전의 광고 소재를 실험하기 어렵다**

   * 광고는 하나의 정답을 만드는 작업이 아니라 여러 소재를 테스트하는 작업이다.
   * 하지만 개인 창작자나 소규모 팀은 소재를 반복 제작할 여력이 부족하다.

---

## 2-2. 핵심 인사이트

사용자가 진짜 원하는 것은 “이미지 한 장 생성”이 아니다.

사용자가 원하는 것은:

> **내 콘텐츠에서 무엇을 광고로 보여줘야 하는지 판단하고,
> 그것을 타겟에 맞는 광고 소재로 빠르게 만들어주는 것**

이다.

즉 HookCut의 경쟁력은 이미지 생성 모델 자체가 아니라,
**콘텐츠 분석 → 광고 기획 질문 → 스토리라인 선택 → 소재 형식 결정 → 여러 광고안 생성** 까지 이어지는 제품 경험이다.

---

# 3. What

## 3-1. 서비스 개요

HookCut은 콘텐츠 원본을 입력받아 광고 소재를 생성하는 서비스다.

입력 가능한 자료:

* 소설 본문
* 작품 소개문
* 위키 URL
* 캐릭터/세계관 설정 문서
* 사용자가 직접 작성한 추가 요청
* 여러 자료의 조합

생성 가능한 결과:

* 광고 배너
* 2컷 분할 광고
* 4컷 만화형 광고
* SNS 카드형 광고
* 컷별 이미지 프롬프트
* 광고 카피
* 타겟별 소재 버전

---

# 4. 핵심 차별점

## 4-1. 범용 이미지 생성기와의 차이

기존 이미지 생성 서비스는 대체로 이렇게 작동한다.

> 사용자가 프롬프트 입력 → 이미지 생성 → 마음에 안 들면 다시 프롬프트 수정

이 방식은 사용자가 광고 기획을 이미 알고 있다는 전제를 가진다.

HookCut은 다르다.

> 콘텐츠 입력 → AI가 광고 포인트 분석 → 필요한 질문만 선택지로 제시 → 사용자가 클릭 몇 번으로 결정 → 광고 소재 생성

즉 HookCut은 **프롬프트 작성 도구** 가 아니라
**광고 소재 생성 워크플로우** 다.

---

## 4-2. HookCut만의 특장점

### 1. Contents-to-Ads에 특화

목적이 명확하다.

> “이미지를 만들고 싶다”가 아니라
> **“이 콘텐츠를 광고로 만들고 싶다”**

에 최적화되어 있다.

### 2. 광고 기획 질문을 자동 생성

사용자가 처음부터 모든 정보를 입력하지 않아도 된다.

AI가 콘텐츠를 먼저 분석한 뒤,
광고 제작에 필요한 질문을 체크리스트로 제시한다.

### 3. 스토리라인 선택 과정 포함

AI가 콘텐츠에서 광고화 가능한 스토리라인 후보를 뽑고,
사용자가 그중 어떤 방향을 소재화할지 선택한다.

예:

* 복수의 시작
* 주인공의 각성
* 관계 역전
* 금지된 사랑
* 세계관의 비밀
* 압도적 능력 등장
* 웃긴 상황/밈 포인트

### 4. 광고 타겟팅을 내장

같은 콘텐츠라도 타겟에 따라 광고가 달라진다.

HookCut은 타겟 정보를 입력받아
카피, 이미지 톤, 장면 선택, 컷 구성을 다르게 만든다.

### 5. 여러 실험 버전 생성

광고는 정답 하나가 아니다.

HookCut은 기본적으로 여러 버전을 생성한다.

예:

* 안정적 버전 1개
* 후킹 강한 버전 1개
* 실험적 버전 1개
* 특정 타겟 집중 버전 1개

---

# 5. 핵심 사용자

## 5-1. 1차 타겟

* 웹소설 작가
* 웹소설 번역자
* 소규모 콘텐츠 운영자
* SNS 홍보 담당자
* 캐릭터/IP 기반 콘텐츠 제작자

## 5-2. 이들이 원하는 것

* 긴 콘텐츠를 짧고 강한 광고 소재로 바꾸고 싶다
* 광고 문구와 이미지를 직접 고민하기 어렵다
* 여러 광고안을 빠르게 뽑아보고 싶다
* 타겟 독자에 맞는 소재를 만들고 싶다
* 디자이너나 마케터 없이도 최소한의 홍보 소재를 만들고 싶다

---

# 6. 핵심 플로우

## 전체 플로우 요약

1. 콘텐츠 자료 입력
2. AI가 콘텐츠 분석
3. 광고화 가능한 스토리라인 후보 제안
4. 광고 타겟/목적/톤 질문
5. 출력 형식 선택
6. 생성 전략 확인
7. 광고 소재 생성
8. 결과 확인 및 재생성

---

## Step 1. 콘텐츠 입력

사용자는 하나 이상의 자료를 입력한다.

### 입력 방식

* 텍스트 직접 입력
* URL 입력
* 문서 업로드
* 추가 요청 프롬프트 입력

### 입력 예시

* 소설 1~5화 본문
* 작품 위키 URL
* 캐릭터 설정 문서
* 세계관 설명
* “강한 후킹형 광고로 만들고 싶다”
* “여성향 로맨스 독자에게 맞추고 싶다”

### 화면 구성

**화면 1: 콘텐츠 입력 페이지**

구성 요소:

* 서비스 설명
* 입력 탭

  * 텍스트 입력
  * URL 입력
  * 문서 업로드
* 추가 요청 입력창
* “분석 시작” 버튼

---

## Step 2. AI 콘텐츠 분석

입력된 자료를 바탕으로 AI가 광고 제작에 필요한 핵심 정보를 추출한다.

### 분석 항목

* 작품 장르
* 주요 인물
* 핵심 갈등
* 초반 후킹 요소
* 감정 포인트
* 독자 기대 요소
* 광고화 가능한 장면 후보
* 피해야 할 스포일러 가능성

### 출력 예시

```text
작품 장르: 회귀 판타지 / 복수극
핵심 갈등: 배신당한 주인공이 과거로 돌아와 복수를 준비함
후킹 포인트: 죽음 직전의 배신, 회귀, 압도적 반격
감정 포인트: 분노, 통쾌함, 기대감
광고 후보: 
1. 배신당하는 순간
2. 회귀를 깨닫는 순간
3. 첫 번째 복수 대상과 마주치는 순간
```

### 화면 구성

**화면 2: 콘텐츠 분석 결과 페이지**

구성 요소:

* AI 분석 요약 카드
* 작품 핵심 훅
* 주요 캐릭터
* 광고 후보 장면
* “다음: 광고 방향 선택” 버튼

---

## Step 3. 광고화할 스토리라인 선택

이 단계가 중요하다.
HookCut은 곧바로 이미지를 만들지 않는다.

먼저 AI가 **광고로 승화할 수 있는 스토리라인 후보** 를 제안한다.

### 스토리라인 후보 예시

#### 후보 A. 복수 후킹형

* 초점: 주인공이 배신당하고 회귀하는 장면
* 장점: 클릭 유도 강함
* 적합 타겟: 판타지/회귀물 독자
* 위험: 흔한 회귀물처럼 보일 수 있음

#### 후보 B. 관계 역전형

* 초점: 주인공을 무시하던 인물이 뒤늦게 후회하는 장면
* 장점: 감정 몰입 강함
* 적합 타겟: 후회물/집착물 선호 독자
* 위험: 작품의 실제 톤과 다르면 과장될 수 있음

#### 후보 C. 세계관 미스터리형

* 초점: 주인공만 알고 있는 세계의 비밀
* 장점: 궁금증 유발
* 적합 타겟: 설정/세계관 중시 독자
* 위험: 즉각적인 자극은 약할 수 있음

### 사용자는 선택한다

* 하나의 스토리라인만 선택
* 여러 개 선택
* AI 추천안 그대로 진행
* 직접 수정 요청

### 화면 구성

**화면 3: 스토리라인 선택 페이지**

구성 요소:

* 스토리라인 후보 카드 3~5개
* 각 카드에 포함:

  * 방향 이름
  * 요약
  * 예상 타겟
  * 장점
  * 주의점
* 선택 버튼
* “AI 추천으로 진행” 버튼

## Step 4. 광고 타겟팅 설정

광고 타겟팅은 MVP에서 너무 복잡하면 안 된다.
대신 광고 소재 품질에 직접 영향을 주는 축만 다룬다.

## 4-1. 타겟팅 축

### 1. 연령대

* 10대
* 20대 초반
* 20대 후반~30대
* 40대 이상
* 전체 대상

영향:

* 카피의 직설성
* 이미지 톤
* 밈/숏폼 감성 사용 여부
* 설명의 친절도

---

### 2. 성별/장르 감성

성별 자체보다 **콘텐츠 소비 성향** 중심으로 묻는 것이 좋다.

선택지:

* 남성향 판타지 독자
* 여성향 로맨스/로판 독자
* 무협/선협 독자
* 라이트노벨/서브컬처 독자
* 일반 대중
* 직접 입력

영향:

* 강조할 캐릭터
* 갈등 표현 방식
* 카피 톤
* 그림체
* 감정선의 강도

---

### 3. 기존 독자 vs 신규 독자

선택지:

* 기존 장르 독자
* 해당 장르를 모르는 신규 유저
* 작품을 이미 아는 팬
* 이탈 유저 재유입

영향:

* 세계관 설명량
* 자극적인 훅 비중
* 캐릭터 이름 노출 여부
* 스포일러 허용 범위

---

### 4. 광고 목적

선택지:

* 작품 첫 클릭 유도
* 첫 화 읽기 유도
* 특정 캐릭터 어필
* 세계관/설정 어필
* 이벤트/프로모션 홍보
* 팬덤 반응 유도

영향:

* CTA 문구
* 컷 구성
* 정보량
* 카피의 구체성

---

### 5. 선호 톤앤매너

선택지:

* 강한 후킹형
* 감성 몰입형
* 캐릭터 중심형
* 코믹/밈형
* 고급스럽고 진지한 톤
* 어두운 분위기
* 밝고 캐주얼한 톤

영향:

* 색감
* 표정
* 구도
* 문장 스타일
* 컷 전개 방식

---

### 6. 피하고 싶은 방향

선택지:

* 과도한 선정성
* 과도한 폭력성
* 스포일러
* 유치한 카피
* 너무 자극적인 낚시성 광고
* 원작과 다른 분위기
* 특정 캐릭터 왜곡

이 항목은 중요하다.
광고 생성 AI는 과장하기 쉽기 때문에,
사용자가 싫어하는 방향을 명시하게 해야 한다.

## Step 5. 출력 형식 선택

초기 기본값은 **2컷 분할 광고 2쌍** 으로 둔다.

즉 기본 생성 결과는:

> 광고안 A: 2컷
> 광고안 B: 2컷

총 4개의 이미지 컷이지만,
“4컷 만화”처럼 하나의 서사를 강제하지 않는다.

이 방식이 더 적합한 이유:

* 광고 배너에 더 가깝다
* 컷 하나하나를 독립 소재로 쓰기 쉽다
* SNS 카드나 배너로 활용하기 좋다
* 4컷 만화보다 부담이 적다
* 여러 버전 실험에 적합하다

---

## 출력 형식 선택지

### 기본값

* **2컷 광고 2쌍**

  * 안정적 광고안 1쌍
  * 실험적 광고안 1쌍

### 선택 가능 형식

* 단일 배너 1장
* 단일 배너 3장
* 2컷 광고 1쌍
* 2컷 광고 2쌍
* 4컷 만화형 광고
* SNS 카드뉴스형 3~5장

### 화면 구성

**화면 5: 출력 형식 선택 페이지**

구성 요소:

* 기본 추천 형식 표시
* 형식 선택 카드
* 생성 개수 선택
* 이미지 비율 선택

  * 1:1
  * 4:5
  * 16:9
  * 9:16
* “생성 전략 확인” 버튼

## Step 6. 생성 전략 확인

이미지 생성 전 마지막으로 전체 전략을 보여준다.

예시:

```text
선택된 콘텐츠 방향:
- 복수 후킹형

- 20대 남성향 판타지 독자

- 첫 화 클릭 유도

톤:
- 강한 후킹형
- 어두운 분위기
- 통쾌한 반격 기대감

- 2컷 광고 2쌍
- 총 4개 이미지
```

사용자는 여기서 다음 중 선택할 수 있다.

* 바로 생성
* 타겟 수정
* 스토리라인 수정
* 출력 형식 수정
* AI 추천으로 다시 구성

## Step 7. 광고 소재 생성

* CTA
* 타겟 설명
* 왜 이 방향으로 만들었는지
* 사용된 스토리라인
* 재생성 버튼

### 결과 예시

* 목적: 첫 화 클릭 유도

* 구성:

  * 1컷: 배신당한 주인공

#### 광고안 B: 실험적 감정 과장형

* 목적: 강한 감정 반응 유도
* 타겟: 통쾌한 복수극 선호 독자
* 카피: “날 버린 자들이, 이제 내 이름 앞에 무릎 꿇는다.”
* CTA: “복수의 시작 보기”
* 구성:

  * 1컷: 조롱받는 과거
  * 2컷: 권력을 쥔 현재

# 7. 유저에게 묻는 질문 설계

## 질문은 자유입력이 아니라 선택형 중심

사용자에게 긴 프롬프트를 요구하면 기존 범용 툴과 다를 바 없다.

# 10. AI 처리 파이프라인

## 1단계: 콘텐츠 구조화

입력 자료를 바탕으로 JSON 형태로 정리한다.

예:

```json
{
  "genre": "회귀 판타지",
  "mainCharacters": ["주인공", "배신자", "조력자"],
  "coreConflict": "배신당한 주인공이 회귀 후 복수를 준비함",
  "emotionalHooks": ["분노", "통쾌함", "기대감"],
  "adStorylineCandidates": [
    {
      "title": "복수의 시작",
      "summary": "배신당한 주인공이 회귀 후 반격을 준비하는 장면",
      "recommendedTarget": "회귀 판타지 독자",
      "risk": "흔한 회귀물처럼 보일 수 있음"
    }
  ]
}
```

## 2단계: 질문 생성

분석 결과를 바탕으로 필요한 질문을 만든다.

```json
{
  "questions": [
    {
      "key": "targetAge",
      "label": "어떤 연령대 독자에게 보여줄까요?",
      "type": "single_select",
      "options": ["10대", "20대", "30대 이상", "전체"]
    },
    {
      "key": "adGoal",
      "label": "광고의 목적은 무엇인가요?",
      "type": "single_select",
      "options": ["첫 클릭 유도", "캐릭터 어필", "작품 분위기 전달"]
    }
  ]
}
```

## 3단계: 광고 전략 생성

사용자 선택값을 바탕으로 광고 전략을 확정한다.

```json
{
  "selectedStoryline": "복수의 시작",
  "target": "20대 남성향 판타지 독자",
  "goal": "첫 클릭 유도",
  "tone": "강한 후킹형",
  "format": "2컷 광고 2쌍",
  "variants": [
    {
      "type": "stable",
      "description": "장르 독자에게 익숙한 회귀 복수 후킹"
    },
    {
      "type": "experimental",
      "description": "감정을 과장한 강한 카피 중심 광고"
    }
  ]
}
```

## 4단계: 광고 소재 생성

최종적으로 광고 이미지 프롬프트와 카피를 생성한다.

```json
{
  "ads": [
    {
      "variant": "stable",
      "headline": "죽었던 날로 돌아왔다",
      "cta": "첫 화 보기",
      "cuts": [
        {
          "caption": "믿었던 동료에게 배신당한 순간",
          "imagePrompt": "dark fantasy scene, betrayed young man..."
        },
        {
          "caption": "눈을 뜨자, 모든 것이 시작되기 전이었다",
          "imagePrompt": "young man wakes up in noble room..."
        }
      ]
    }
  ]
}
```

# 12. 최종 제품 포지셔닝

HookCut은 이미지 생성 모델을 대체하려는 서비스가 아니다.

오히려 기존 이미지 생성 모델 위에 얹히는
**콘텐츠 광고화 워크플로우 레이어** 에 가깝다.

정리하면:

> 범용 이미지 생성기는 “무엇을 그릴지” 사용자가 알아야 한다.
> HookCut은 콘텐츠를 분석해 “무엇을 광고로 만들어야 하는지”부터 도와준다.

그래서 HookCut의 핵심 가치는 다음이다.

* 콘텐츠 이해
* 광고 포인트 추출
* 스토리라인 선택
* 타겟 기반 광고 전략
* 출력 형식 결정
* 여러 광고안 생성

즉 HookCut은
**콘텐츠를 광고 소재로 바꾸는 과정을 제품화한 AI Contents-to-Ads 서비스** 다.
